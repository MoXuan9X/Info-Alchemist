# VOI 原则

VOI = Value of Information。信息不是因为“真实”就有价值，而是因为它可能改变行动。

使用这条核心规则：

```text
如果答案不会改变用户下一步行动，就不要继续消耗搜索预算。
```

必须识别：

- 默认行动：如果现在停止研究，用户最可能做什么
- 候选行动：至少两个可选行动
- 决策边界：用户距离行动改变有多近
- 高 VOI 问题：答案可能改变默认行动的问题

低 VOI 问题：
- “这是什么？”
- “它火不火？”
- “大家在说什么？”

高 VOI 问题：
- “API 是否开放，并且是否允许商业工具使用？”
- “用户是否带着交易或工具意图在搜索？”
- “是否存在一个低成本假门测试，可以验证或证伪这个方向？”
