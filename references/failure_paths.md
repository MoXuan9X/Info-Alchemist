# 失败路径

## 没有联网搜索 Key

如果缺少联网搜索 key，不要假装已经联网搜索。输出搜索计划，并提醒用户补齐联网搜索配置或提供搜索结果。调试场景可检查 `TAVILY_API_KEY`。

## 搜索结果很弱

如果联网搜索返回结果很弱、不相关或证据不足：

- 降低 confidence
- 把缺口标成 `EVPI` 或 `EVSI`
- 优先选择 `watch` 或 `probe`
- 不要选择 `act`

## 用户已经决定

如果用户看起来是在寻找确认信息，把 `decision_boundary.status` 设为 `already_decided`，并主动列出可能证伪这个决定的 `painful_evidence`。

## 高风险领域

医疗、法律、金融或投资决策中，不推荐最终行动。只提供证据整理、风险框架和取证渠道。

## 记忆被禁用

如果记忆被禁用或不安全，可以生成 `memory_update` 提案，但不要写入磁盘。
