# 个人决策画像

Info-Alchemist 的记忆功能用于生成“个人决策画像”。

它的目标不是保存聊天记录，也不是保存搜索到的文章，而是帮你回看：自己是怎么做信息决策的。

## 它是什么

当你看完一份信息炼金报告，并明确告诉 AI Agent 你的最终决定后，Info-Alchemist 可以把这次决策整理成一条本地记录。

多次记录之后，它会形成一份个人决策画像，帮助你看到：

- 你通常因为什么信息开始调研。
- 你原本倾向采取什么行动。
- 哪类证据最容易改变你的判断。
- 你最近做过哪些关键选择。
- 下次遇到类似问题时，应该优先补什么证据。

## 它有什么价值

个人决策画像可以帮助你避免重复低价值调研。

例如，你可能会发现：

- 自己经常因为“看到一个新趋势”就开始查很多资料。
- 真正改变决定的证据通常不是新闻，而是用户反馈、竞品路径、价格、留资或转化数据。
- 有些方向你反复进入“继续观察”，但没有形成最小验证动作。
- 下次遇到类似机会时，可以先问更关键的问题，而不是继续堆链接。

## 它不会保存什么

个人决策画像默认只保存在你的本地电脑。

它不会主动保存：

- 原始文章全文
- 完整聊天记录
- 隐私文件
- 敏感数据
- 你没有明确要求记录的决策

## 什么时候会记录

不会在每次搜索后自动记录。

只有当你明确告诉 AI Agent 你的决定，或要求它记录这次决策时，才会写入本地记忆。

你可以这样说：

```text
记录这次决策：我决定先小范围验证，不直接开发完整产品。
```

## 文件说明

这个目录里的 README 只是说明文档。

正式运行时，个人决策画像和决策记录会保存在本地运行数据目录里，通常在 `info-alchemist/memory/` 下。
