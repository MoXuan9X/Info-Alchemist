连接 Info-Alchemist 的搜索能力

先填写 Tavily API key，Info-Alchemist 才能搜索公开网页。 如果你还想查看小红书、X、Reddit 的用户讨论，再打开社交平台搜索。

用于公开网页搜索，比如官方资料、新闻、竞品页面、定价页和文档。

去哪里获取

打开 Tavily 后注册或登录，在 Dashboard / API Keys 里复制以 tvly- 开头的 key。

打开 Tavily
社交平台搜索 可选

打开后会补充小红书、X、Reddit 的用户讨论、抱怨、口碑和社区信号。

什么时候打开

当你想判断真实用户怎么讨论、吐槽或比较一个产品时再打开。普通公开网页搜索不需要。

用于查小红书、X、Reddit 上的公开讨论和社区信号。

去哪里获取

打开 TikHub 后注册或登录，在控制台、API key 或 Token 页面复制密钥。

打开 TikHub
填写后点击保存。
2保存后怎么做

回到 AI Agent，重新发送刚才的问题。

保存完成后，不需要重装 Info-Alchemist。回到原来的 AI Agent 对话，重新问刚才那个真实问题即可。

  • 1看到第一行出现 INFO_ALCHEMIST=TRUE。
  • 2报告里出现核心判断、证据、缺口和下一步行动。
  • 3需要社交平台信号时，再回来打开社交平台搜索。
本地保存： 配置会写入 本地 .env 文件。这个页面只负责保存搜索配置。
