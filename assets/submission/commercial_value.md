# 信息炼金术士Skill商业价值

目标用户：OPC、超级个体、独立开发者、创业者、产品经理等相关角色

用户痛点：AI 时代信息太多、机会太像、调研越做越焦虑，什么事都可以做，但不知道怎么决策更合适。

解决方案：通过信息炼金术士Skill，先挖掘用户的搜索意图，根据意图创建搜索计划，再联网搜索信息，找出离用户决策更近的信息源，最后运用 VOI 框架把信息证据映射到用户的行动判断。并且在对话过程中，会记录用户的个人决策画像，包括近期决策记录、惯性决策模式、有效证据类型、最近决策洞察和下次询证提醒，让用户知道自己常被什么信号触发，哪些证据真的改变过行动。

付费理由：用户购买的不是一个搜索工具，而是一个信息海关。它减少无效调研、错误投入和追热点冲动，让每条信息都有去向，让焦虑变成证据，让行动变成可复盘实验。
