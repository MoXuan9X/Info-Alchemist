# 3 分钟路演稿

## 0:00 - 0:30 痛点

AI 时代，每天都有新模型、新 API、新工具站、新竞品。普通用户看到这些信息，会本能地想：这个有没有机会？我要不要跟？但问题是，普通搜索会给你更多链接、更多观点、更多焦虑，却不告诉你下一步该不该行动。

## 0:30 - 0:55 产品介绍

信息炼金术士不是普通搜索助手。它的核心是 VOI，Value of Information。它先问：这条信息会不会改变你的行动？如果不会，就不要继续消耗注意力。如果会，再做联网搜索取公开证据。

## 0:55 - 1:50 Demo 输入和决策澄清

用户输入：帮我查一下 AI 图片工具站有哪些竞品。

普通搜索会立刻查资料。信息炼金术士先澄清真实决策：你查 AI 图片工具站是为了什么，确认用户是在决定要开发工具站还是找好用的工具站自己使用。它列出候选行动：直接开发完整工具站、先做对比页、只观察、暂时不做。然后识别默认行动：如果不处理，你大概率会继续研究。当前状态是接近决策，已经接近投入，但还缺行动证据。

## 1:50 - 2:25 公开证据和输出

接着它生成联网搜索计划：查官方 API 和价格、查竞品和变现、查用户讨论、查教程和榜单页搜索意图。联网搜索负责公开证据，Skill 负责判断这些证据会不会改变行动。输出不是资料堆叠，而是公开证据、痛感证据和证据缺口。比如官方能力能降低技术风险，但搜索需求、点击意图、愿付信号和 API 单位成本，仍然不能靠公开搜索直接回答。

## 2:25 - 2:45 记忆

最后它给出最终状态：小范围验证。下一步不是开发完整工具站，而是先做轻量对比页或榜单页，加体验按钮或等候名单。它还生成记忆更新，只记录派生洞察：这次触发可能来自热点焦虑，真正改变行动的是搜索意图和低成本转化信号。

## 2:45 - 3:00 收束金句

普通搜索记住你查过什么。信息炼金术士记住什么信息改变过你，什么信息只是骗走了你。焦虑来自：每条信息都像机会。安心来自：每条信息都有去向。
